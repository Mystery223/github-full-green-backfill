# GitHub FULL GREEN Contribution Generator

Tool ini akan membuat contribution FULL HIJAU selama 30 hari terakhir
(8-15 commit per hari).

## Cara Pakai

1. Extract zip ini
2. Masuk ke repository GitHub kamu
3. Copy file `full_green.py` ke dalam repo
4. Jalankan:

```
python3 full_green.py
```

5. Setelah selesai:

```
git push origin main
```

## Penting

Pastikan email git sesuai dengan akun GitHub:

```
git config user.email
```

Kalau belum:

```
git config --global user.email "emailkamu@gmail.com"
```
